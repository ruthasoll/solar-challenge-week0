## Objective
The objective of this project is to get everyone comfortable with version control using Git and GitHub before working with data.

## Environment Setup

### 1. Initialize Repository

1. **Create a New GitHub Repository**:
   - Go to GitHub and create a new repository named `solar-challenge-week0`.

2. **Clone the Repository Locally**:
   - Open your terminal and navigate to the directory where you want to clone the repository.
   - Run the following command:
     ```bash
     git clone https://github.com/ruthasoll/solar-challenge-week0.git
     ```
   - Replace `<your-username>` with your GitHub username.

3. **Set Up a Python Virtual Environment**:
   - Navigate into your cloned repository:
     ```bash
     cd solar-challenge-week0
     ```
   - Create a virtual environment using either `venv` or `conda`:

   **Using venv**:
   ```bash
   python -m venv venv
